Refactoring

/v1/book/all  GET      | Acessa todos os livros de algum usuário
/v1/book      POST     | Cadastra um livro para o usuário
/V1/book/id   GET      | Acessa o livro de id tal
/v1/book/id   DELETE   | Deleta o livro de id tal
------------------------
/v1/user/all  GET      | Acessa todos os usuários cadastrados      | OK
/v1/user      POST     | Cadastra um novo usuário no sistema       | OK
/v1/user/id   DELETE   | Deleta o usuário de id tal                | OK
------------------------
/token        GET      | Gera um token de acesso para o usuário    | OK
